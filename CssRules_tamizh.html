CSS Selectors: HTML, CSS, and Output

Universal Selector (*)


/* CSS */
* {
    border: 1px solid red;
    margin: 5px;
    padding: 5px;
}

/* HTML */
<div>All elements will have red border</div>
<p>This includes paragraphs, divs...</p>
All elements will have red border

This includes paragraphs, divs...

Type Selector


/* CSS */
p {
    color: green;
}

/* HTML */
<p>This paragraph will be green</p>
<div>This div will not be affected</div>

This paragraph will be green

This div will not be affected

Class Selector


/* CSS */
.highlight {
    background-color: yellow;
    font-weight: bold;
}

/* HTML */
<p class="highlight">Yellow background</p>
<div class="highlight">Also yellow</div>
<p>Not highlighted</p>

Yellow background

Also yellow

Not highlighted

ID Selector


/* CSS */
#main-title {
    color: blue;
    font-size: 24px;
    text-align: center;
}

/* HTML */
<h1 id="main-title">This title is blue</h1>
<h2>Only one element can have this ID</h2>

This title is blue

Only one element can have this ID

Attribute Selector


/* CSS */
input[type="text"] {
    border: 2px solid purple;
    padding: 5px;
}

input[required] {
    background-color: lightyellow;
}

/* HTML */
<input type="text" placeholder="Purple border">
<input type="email" placeholder="Normal border">
<input type="text" required placeholder="Yellow background">

Pseudo-Class Selectors


/* CSS */
a:hover {
    color: red;
    text-decoration: underline;
}

input:focus {
    outline: 2px solid green;
    background-color: lightyellow;
}

/* HTML */
<a href="#">Hover over me</a>
<input type="text" placeholder="Click me">
Hover over me

:first-child Selector


/* CSS */
li:first-child {
    color: orange;
    font-weight: bold;
}

/* HTML */
<ul>
    <li>First item (orange)</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>
  • First item (orange)
  • Second item
  • Third item

:nth-child Selector


/* CSS */
tr:nth-child(even) {
    background-color: #f2f2f2;
}

/* HTML */
<table>
    <tr><td>Row 1</td></tr>
    <tr><td>Row 2 (highlighted)</td></tr>
    <tr><td>Row 3</td></tr>
    <tr><td>Row 4 (highlighted)</td></tr>
</table>
Row 1
Row 2 (highlighted)
Row 3
Row 4 (highlighted)

Descendant Selector


/* CSS */
.container p {
    color: blue;
}

/* HTML */
<div class="container">
    <p>Direct paragraph (blue)</p>
    <span>
        <p>Nested paragraph (also blue)</p>
    </span>
</div>

Direct paragraph (blue)

Nested paragraph (also blue)

Child Selector (>)


/* CSS */
.parent > .child {
    border: 2px solid green;
}

/* HTML */
<div class="parent">
    <div class="child">Direct child (green border)</div>
    <span>
        <div class="child">Nested child (no border)</div>
    </span>
</div>
Direct child (green border)
Nested child (no border)

Adjacent Sibling Selector (+)


/* CSS */
h2 + p {
    font-style: italic;
}

/* HTML */
<h2>Heading</h2>
<p>Directly after heading (italic)</p>
<p>Another paragraph (normal)</p>

Heading

Directly after heading (italic)

Another paragraph (normal)

General Sibling Selector (~)


/* CSS */
h2 ~ p {
    color: purple;
}

/* HTML */
<h2>Heading</h2>
<p>First paragraph (purple)</p>
<p>Second paragraph (purple)</p>
<span>Not affected</span>

Heading

First paragraph (purple)

Second paragraph (purple)

Not affected
